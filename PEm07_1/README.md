# Анализатор блюд на основе YandexGPT Vision

CLI приложение для анализа фотографий блюд с использованием YandexGPT Vision API. Определяет название блюда, ингредиенты, тип кухни и уровень уверенности.

## Возможности

- 🍽️ Определение названия блюда по фотографии
- 🥘 Список предполагаемых ингредиентов
- 🌍 Определение типа кухни
- 📊 Уровень уверенности в определении

## Требования

- Python 3.10 или выше
- API ключ YandexGPT
- ID каталога в Yandex Cloud

## Установка

1. Клонируйте репозиторий или скачайте файлы проекта

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

4. Заполните `.env` файл своими данными:
   - `YANDEX_GPT_API_KEY` - ваш API ключ YandexGPT
   - `YANDEX_GPT_FOLDER_ID` - ID каталога в Yandex Cloud

## Получение API ключа

1. Зарегистрируйтесь в [Yandex Cloud](https://cloud.yandex.ru/)
2. Создайте платежный аккаунт
3. Создайте каталог (folder) в Yandex Cloud
4. Создайте сервисный аккаунт с ролью `ai.languageModels.user`
5. Получите API ключ для сервисного аккаунта
6. Скопируйте ID каталога из настроек

Подробные инструкции: [Документация YandexGPT](https://yandex.cloud/ru/docs/foundation-models/quickstart/yandexgpt)

## Использование

### Базовое использование

```bash
python main.py путь/к/изображению.jpg
```

### Сохранение результата в файл

```bash
python main.py dish.jpg -o result.json
```

## Формат вывода

Приложение возвращает результат в формате JSON:

```json
{
  "dish_name": "Паста карбонара",
  "ingredients": [
    "спагетти",
    "бекон",
    "яйца",
    "пармезан"
  ],
  "cuisine": "итальянская",
  "confidence": 0.82
}
```

## Поддерживаемые форматы изображений

- JPEG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)
- WebP (.webp)

## Примеры

```bash
# Анализ изображения с выводом в консоль
python main.py pasta.jpg

# Анализ с сохранением результата
python main.py sushi.jpg -o analysis_result.json
```

## Обработка ошибок

Приложение обрабатывает следующие ошибки:
- Отсутствие файла изображения
- Неверный формат изображения
- Проблемы с API ключом
- Ошибки сети
- Некорректный ответ от API
